# Group-4-Unit-1-Project

A basic countdown timer web app built with HTML, CSS, and Javascript. It starts from 30 minutes and lets you start, pause, and reset the timer.

How to use it?

Open the index.html file in your web browser.
Click the start button to begin the timer countdown.
Click the pause button to pause the countdown at any time.
Click the reset button to reset the timer back to 30 minutes.
